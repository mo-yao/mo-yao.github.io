Markdown语法支持的Emoji表情

前言

Markdown 🏃 语法支持添加 emoji 表情喔 😋，快来试试吧 ~ 💓💓💓

通用代码 Unicode

&#x + unicode + ;

例如：&#x1F601;

  2. U+1F495

  6. U+1F647
  7. U+1F9DC
  8. U+1F69A
  9. U+1F6A2
  10. U+1F383
  11. ……

Emoji表情速查

: + 对应英文单词 + :
例如：:smile:

  1. :+1:   ; :-1:
  2. :ear:   ; :eyes:
  3. :feet:  ; :lips:
  4. :poop:   ;:shit:
  5. :angry:  ; :rage:
  6. :fu:   ;:walking:
  7. :star2:   ;:dizzy:
  8. :fire:  ; :hankey:
  9. :kiss:   ; :droplet:
  10. :bowtie:  ; :smile:
  11. :muscle:  ; :metal:
  12. :sparkles:  ; :star:
  13. :wink:  ; :grinning:
  14. ......
