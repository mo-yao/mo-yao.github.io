推荐几个你可能会用到的 CSS 动画库

发表于2023-08-16|更新于2023-08-16|CSS
|字数总计:193|阅读时长:1分钟|阅读量:

Animate.css

Animate.css 是一个即用型跨浏览器动画库。

OBNOXIOUS.CSS

OBNOXIOUS 的动画效果，有亿点点意思。

CSShake

传送门：https://elrumordelaluz.github.io/csshake/

AnimXYZ

AnimXYZ 可帮助你的网站创建、自定义动画。由 CSS 变量提供支持，无需编写单个关键帧即可实现几乎无限数量的独特动画。

Aimista

Animista 是一个 CSS 动画库，您可以在其中使用一系列现成的 CSS 动画并仅下载您将使用的动画。

Hover.css

Hover.css 鼠标悬停效果集合，可应用于链接、按钮、徽标、SVG、特色图像等。轻松应用到您自己的元素、修改或仅用于获取灵感。

Magic CSS

Magic CSS 是一组简单的动画，可包含在 Web 或应用程序项目中。
