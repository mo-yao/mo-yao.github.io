Emoji表情速查

: + 对应英文单词 + :
例如：:smile:

  1. :+1: &nbsp; ; :-1:
  2. :ear: &nbsp; ; :eyes:
  3. :feet:&nbsp; ; :lips:
  4. :poop: &nbsp; ;:shit:
  5. :angry:&nbsp; ; :rage:
  6. :fu: &nbsp; ;:walking:
  7. :star2: &nbsp; ;:dizzy:
  8. :fire:&nbsp; ; :hankey:
  9. :kiss: &nbsp; ; :droplet:
  10. :bowtie:&nbsp; ; :smile:
  11. :muscle:&nbsp; ; :metal:
  12. :sparkles:&nbsp; ; :star:
  13. :wink:&nbsp; ; :grinning:
  14. ......
